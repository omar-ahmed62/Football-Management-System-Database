# Football Tournament Management System

# Description:
A SQL Server database project for managing football tournament (e.g., FIFA World Cup), teams, players, matches, referees and stadiums

# Technologies:
- SQL SERVER MANGMENT STUDIO (SSMS)
- T SQL
- GIT
- GITHUP
